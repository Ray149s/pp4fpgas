# Labs

Lab assignments and project materials for the pp4fpgas course (UCSD CSE 237C).

## Directory Structure

| Directory | Description |
|-----------|-------------|
| `Lab1/` | Vivado HLS tutorial labs covering arbitrary precision, C validation, design analysis, optimization, interface synthesis, and RTL verification |
| `Vivado_HLS_Tutorial/` | Full Vivado HLS tutorial with documentation and examples |
| `project1/` | FIR filter implementations and Vivado tutorial PDFs |
| `project2/` | CORDIC and phase detector HLS code |
| `project3/` | DFT implementations |
| `project4/` | OFDM receiver project |
| `axi4_burst/` | AXI4 burst interface lab |
| `cordic/` | CORDIC algorithm lab (Demo and HLS) |
| `DFT/` | DFT lab with precomputed variants |
| `FFT/` | FFT lab with HLS implementations |
| `phase_detector/` | Phase detector lab (Demo, HLS, Simulink) |
| `images/` | Lab-specific images |

## Running Labs

Labs require Xilinx Vivado HLS (now Vitis HLS). Refer to the individual lab directories for specific instructions and TCL scripts.
